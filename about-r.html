About R

R is a top ranking tool for data scientists and analysis teams in many industries. It is a cross-platform programming language and software environment for statistical computing and data analysis. It is free and open source (download here), and has a large and very active community.

It is very flexible and extensible, and supports integration with a variety of programming languages (e.g. C, C++, JavaScript, ...) and has a suite of tools for producing reproducible research, automated reports, and modern data-driven web tools.
